# Getting the Data

You need shopping data for the chatbot to answer questions.

## Steps

1. Go to this website: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce/
2. Click the big Download button
3. Unzip the downloaded file
4. Create a folder called `data` using 'mkdir data'
5. Put all the CSV files from the unzipped folder into the `data` folder

## Check

Run `ls data/` to see if you have 9 CSV files.

## Next

After this, run `python data_setup.py` to load the data.
